# vendospace.com
Find the best adspace for your Out-Of-Home campaign

## Built with
* nextjs
* nodejs
* express
* mongodb

## Get Started
### Prerequisite
    node 16.16.0
    npm 8.11.0

### Installation
    mkdir vendospace.com &&\
    cd vendospace.com &&\
    git clone git@github.com:fdvmoreira/vendospace.com.git .

### Usage
Inside vendospace.com directory, type the following command:
``` 
cd app &&\
npm install &&\
npm run server
 ```
